title: 评论系统配置
---

::alert{type="info"}
在评论系统配置 `comment` 下目前共有 **2** 个子项 ( `enable`、`twikoo`)
::

::alert{type="warning"}
目前评论系统仅支持twikoo，后续会增加对其他系统的支持
::

## twikoo 配置
参考 [twikoo文档](https://twikoo.js.org)

```yaml [_config.acrylic.yml]
twikoo:
    envId: https://twikoo.nalex.top
```
